# Editor & admin polish

Editor polish and admin quality-of-life updates for block editing, exports, and admin workflows.

### Features

- **Block editor:** Table block supports column alignment and header row toggle.
- **Document export:** PDF export queue shows progress in the document actions menu.
- **Help hub:** Cross-links from What's new to Help & guides for onboarding flows.

### Fixes

- Publish action disabled correctly when the user has write but not publish permission.
- Department context breadcrumb no longer duplicates the company name on narrow viewports.
- Notification email digest respects per-channel opt-out without clearing in-app alerts.

### Performance

- Document version list loads incrementally for documents with more than 50 revisions.
- MinIO attachment metadata is cached for the duration of an editing session.

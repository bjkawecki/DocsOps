# Notifications & What's new

This release focuses on staying informed about document changes and smoother team workflows.

### Features

- **What's new page:** Release notes at `/whats-new` with version cards and section icons for Features, Fixes, and Performance.
- **In-app notifications:** Bell icon in the header with unread count; filter by document changes, draft requests, and system messages.
- **Recent items:** Scope-aware recent panel remembers last visited processes, projects, and documents.

### Fixes

- Search snippets no longer break layout when Markdown bold appears inside highlighted segments.
- Account menu version label stays visible in collapsed sidebar mode.
- Restore from backup no longer leaves maintenance mode stuck after a failed upload step.

### Performance

- Catalog search uses improved FTS headline rendering for faster perceived load on large result sets.

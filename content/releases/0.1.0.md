# Initial internal release

Welcome to DocsOps – your internal documentation platform for teams, departments, and company-wide knowledge.

### Features

- **Organisation:** Company, department, and team structure with scoped contexts (process, project, personal space).
- **Documents:** Markdown and block-based editing, explicit read/write permissions, publish workflow, and version history.
- **Admin & operations:** User administration, disaster-recovery backup (PostgreSQL + MinIO), and restore from the admin UI.
